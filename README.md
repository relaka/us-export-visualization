OVERVIEW
- This Project uses Python, Tkinter, and Matplotlib to visualize U.S. agriculture export data 
- This project provides charts for October 2024/2025 and November 2024

The charts include 

October:
- Soybeans: Horizontal stacked bar chart
- Corn: Vertical stacked bar chart
- Wheat: Line chart

November:
- Soybeans: Pie chart 
- Corn: Horizontal bar chart
- Wheat: Area Chart

The charts can be viewed with a simple GUI made with Tkinter with buttons to visualize charts 

-----------------------

REQUIREMENTS

- Install the following Python Libraries using pip (copy and paste the line of code below if you do not have them installed) 
    pip install matplotlib pandas numpy

- Tkinter should come installed with most python but if not use the line of code below to install 
    pip install tk
-----------------------

HOW TO RUN THE PROGRAM

1. Clone the Repository 

2. Navigate to the Project folder 

3.  Open the python script named main.py and run it using the following command below 
    python main.py
    You may need to use: python3 main.py

4. The Tkinter GUI will open 
    Click the button that correlates with each chart to open the chart

------------------------

ASSUMPTIONS

- November data only includes 2024 data, so charts for November only visualize one year 
- The GUI displays in a sepearte Matplotlib window which will open when the corresponding button is selected 
- Data provided in the spreadsheet is used directly
